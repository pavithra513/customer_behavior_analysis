
# Customer Behavior Analysis


Data analytics project showcasing customer behavior analysis using python ,sql and PowerBI.


## DataSet Overview

The dataset Overview This project demonstrates an end-to-end data analytics workflow, including data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, and dashboard visualization. The goal is to extract meaningful insights from raw data and present them in a clear, business-friendly format.
## Tools & Technology

Python (Pandas, NumPy, Matplotlib, Seaborn)

PostgreSQL (for data storage and querying)

Power BI (for interactive dashboard creation)

## Project Steps
## Data Loading

Imported the dataset into Python using Pandas.
Performed initial inspection to understand structure and data types.
Exploratory Data Analysis (EDA)
Analyzed distributions, correlations, and patterns.
Identified trends, anomalies, and potential issues in the dataset.
Visualized key insights using plots and charts.

## Data Cleaning

Handled missing values and duplicates.
Corrected data types and standardized formats.
Removed or treated outliers where necessary.
SQL Analysis (PostgreSQL)
Loaded cleaned data into PostgreSQL.
## PowerBI Dashboard

Connected Power BI to PostgreSQL database.

Built interactive dashboards with:

Filters and slicers
Visual representations (charts, graphs)
Dashboards

Interactive Power BI dashboard showcasing:

## Running Steps

Clone the Repository

git clone <your-repo-link>
cd <project-folder>
Set Up Python Environment

pip install -r requirements.txt
Run Data Processing & EDA

python main.py
Set Up PostgreSQL

Create a database
Import cleaned dataset
Run SQL scripts provided in /sql folder
Open Power BI Dashboard

Open .pbix file in Power BI
Connect to your PostgreSQL database
Refresh data to view dashboards
